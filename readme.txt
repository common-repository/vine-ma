Vine is a Marketing automation tool that helps you generate better leads from your web site visitors and gradually convert them into buying customers. It includes Web site visitor tracking , web forms (including interactive bot forms), landing pages, AI chatbot (including live chat) , email marketing, automated workflows, thorough analytics, and integration with Vine CRM or third party tools.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'vine'
3. Activate Vine from your Plugins page
4. Login by using MA account at the plugin settings page
